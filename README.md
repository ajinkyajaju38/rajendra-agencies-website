# Rajendra Agencies — Website

## 🚀 Quick Deploy to Netlify

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → New Site → Import from GitHub
3. Select this repo → Deploy
4. Done! Site is live at `yoursite.netlify.app`

## 🌐 Custom Domain Setup

1. Buy domain at GoDaddy / Namecheap / BigRock
2. Netlify Panel → Site Settings → Domain Management → Add Custom Domain
3. In your domain provider DNS settings:
   - CNAME | www | yoursite.netlify.app
   - A Record | @ | 75.2.60.5
4. SSL auto-activates within minutes ✅

## ✏️ Client Content Editing (Decap CMS)

1. In Netlify: Enable Identity + Git Gateway (Site Settings → Identity → Enable)
2. Go to `yoursite.netlify.app/admin`
3. Log in → Edit products, contact info, images
4. Save → Site auto-rebuilds in ~30 seconds
